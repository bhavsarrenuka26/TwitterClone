# 🐦 Twitter (X) UI Clone

A responsive, static frontend web application replicating the user interface of Twitter (X). This project was built to practice modern web development layouts and utility-first CSS styling.

🔗 **Live Demo:** https://twitteruibyrenuka.netlify.app/

## 🛠️ Tech Stack
* **HTML5:** Semantic markup for page structure.
* **Tailwind CSS:** Utility-first CSS framework for rapid UI styling and responsive design.
* **Vite:** Used for a fast local development server.

## ✨ Features
* Responsive layout that adapts to different screen sizes.
* Left navigation sidebar with hover effects.
* Central main feed for scrolling through posts.
* Right sidebar for search and trending topics.
* Modern, clean UI replicating the actual platform.

## 🚀 How to Run Locally

If you want to download this code and run it on your own machine, follow these steps:

1. **Clone the repository:**
 
   git clone [https://github.com/bhavsarrenuka26/TwitterClone.git](https://github.com/bhavsarrenuka26/TwitterClone.git)

2.  Navigate into the project folder:

      cd TwitterClone.
    
      Install dependencies:
    
      npm install
4.  Run the development server and Tailwind compiler:

    To start the Vite server: npm run dev
    To watch for Tailwind CSS changes: npm run dev:css
